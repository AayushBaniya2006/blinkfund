# Tailark Component Installation Commands

All commands to install Tailark components (excluding mist components):

## Call to Action Components
```bash
pnpm dlx shadcn add @tailark/call-to-action-1 --overwrite --yes
pnpm dlx shadcn add @tailark/call-to-action-2 --overwrite --yes
pnpm dlx shadcn add @tailark/call-to-action-3 --overwrite --yes
```

## Comparator Components
```bash
pnpm dlx shadcn add @tailark/comparator-1 --overwrite --yes
```

## Quick Install (All at once)
Run the installation script:
```bash
./scripts/install-tailark-components.sh
```
